Why?

For fun! To monitor rainfall and temperatures. To learn about raspberry pis, and as a prototype for a 3G connected weather station in remote areas.

This weather station is located in Atherton, CA

Instrument

Hardware

The weather station consists of a Raspberry Pi 3 and a set of sensors.

Computer

A Raspberry Pi Zero provides recording and transmission from the weather station. On average, it uses 100mA at 5V.

Solar panel

A 25W solar panel provides power to recharge the battery. The panel cost $38, but I bought a $39 mounting bracket, so the solar ended up being the most expensive part of the project. The 12V of the panel is monitored with a AdaFruit INA219 High side DC current monitor. This lets me monitor how much power the panel is collecting.

Battery

A 3.3Ah 12V lead acid battery provides power for the pi zero. The battery cost $15. I previously tried a Li-ion battery but the charge controler prevented the battery from switching between charging and outputting smoothly. The battery is charged using a $10 solar panel charge controler .

Thermometer, barometer and hygrometer

I've used a conveniently packaged Adafruit BME280 sensor. Currently, this is $20, which is not the cheapest, but the convienece of having all three in one package won me over.

Rain Gauge

I constructed an inexpensive tipping rain gauge from an empty coconut oil jar, a empty juice jar, threaded rod, brass tube and a funnel. Each tip recorded using a reed switch connected to the GPIO header on the pi zero.




Software

Data is collected using an I2C protocol, which allows multiple devices to operate simultaneously.

A python script is run every 5 minutes collecting temperature, pressure and humididty data. The rain gauge is monitored continuously. A picture is taken every day at noon. Code can be found on my github repo:https://github.com/jetolan/weather/.

A cron job controls the transfer of the csv files and image files to an AWS EC2 instance, which hosts the web with a simple static apache server. The website is updated every 10 minutes using a cron job.

The web display employees the python package bokeh, which conveniently converts python figures into interactive web javascript objects.
