Warner Valley Weather Station

Why?

Since the road is closed from Chester, it is nice to know what is going on in the valley!

The weather station is located above Lee Camp

Instrument

Hardware

The weather station consists of a Raspberry Pi 3 and a set of sensors.

Software

Data is collected using an I2C protocol, which allows multiple devices to operate simultaneously.
